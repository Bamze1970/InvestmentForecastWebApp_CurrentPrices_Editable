# Investment Forecast Web App – Current Prices Edition

Това е PWA / GitHub Pages версия с отделна секция **Актуални цени**.

## Какво има вътре
- текуща цена за всяка инвестиция
- предишна цена за всяка инвестиция
- разлика
- стрелка:
  - зелена нагоре
  - червена надолу
  - сива без промяна
- ръчно обновяване на цените
- локално запазване в браузъра
- автоматично преизчисляване на 1 месец, Q1, Q2, Q3, Q4, 1Y, 2Y, 3Y, 4Y

## Как работи
При натискане на „Запази“ новата цена става текуща, старата текуща цена става предишна и приложението веднага обновява всички бъдещи калкулации.

## Качване в GitHub Pages
1. Разархивирай файла.
2. Качи всички файлове в root-а на нов GitHub repository.
3. Включи Pages от main branch и /(root).
4. Отвори линка и инсталирай като приложение.
